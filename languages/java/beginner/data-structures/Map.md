# Map/Dictionary

## Definition

- Collection, implements Map interface
- Store key and value pairs
- when defined in type, <Key, Value>
  - both can be nulls
  - keys must be unique
- unordered

## Methods

- put(key, value)
- get(key)
- keySet()
- values()
- remove(key)
- entrySet()
