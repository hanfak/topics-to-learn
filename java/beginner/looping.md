### Looping/Iterating
  - Repeating the same task
  - Doing the same thing across a data structure
  - ```for```/```while```/```do while``` (imperative)
    - New for each loop
      - ```for (String element : listOfStrings) {...}```
  - ```continue```
  - ```break```
  - Recursion
  - Nested loops
